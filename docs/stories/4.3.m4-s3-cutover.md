## Architecture: Before → After

### Legacy (n8n)
```
Telegram → n8n Webhook
  → Query Supabase
  → Claude node
  → Save progress
  → Send reply
```

### Current (FastAPI)
```
Telegram → FastAPI Webhook
  ├─ IntakeService (me-testa)
  ├─ MeTestaAnswerService
  ├─ SocraticoService
  ├─ ApkgBuilderService
  └─ WeeklyReportJobService
    ↓
  Supabase (persistent)
  ↓
  Response → Telegram
```

## Performance Gains
- n8n startup: ~30s → FastAPI: ~2s
- Memory: n8n ~500MB → FastAPI ~50MB
- Latency: API call + node execution → direct function call
