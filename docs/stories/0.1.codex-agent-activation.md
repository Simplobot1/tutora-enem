# Story 0.1: Codex Agent Activation

## Status
Done

## Story
**As a** operador do Codex CLI,
**I want** ter todos os agentes AIOX disponíveis como skills locais do Codex,
**so that** eu possa ativar qualquer persona diretamente no repositório sem configuração manual adicional.

## Acceptance Criteria
1. A pasta `.codex/skills` existe no repositório com um skill `aiox-<agent-id>` para cada agente em `.aiox-core/development/agents/`.
2. Cada skill do Codex referencia o arquivo canônico do agente e o comando oficial de greeting.
3. A validação dos skills do Codex passa sem erros.
