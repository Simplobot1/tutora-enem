# ✅ AIOX Agents — Forma Simples

## 🎯 Funciona Assim (Simplesmente)

No seu prompt, escreva **`@nomeDoPapel`** e o agente é ativado:

### Dev (Desenvolvimento)
```
@dev
Implementar login com Telegram
```

### QA (Testes)
```
@qa
Rodar QA Gate nessa story
```

### Architect (Arquitetura)
```
@architect
Como devemos fazer o RLS do Supabase?
```

### Scrum Master (Criar Stories)
```
@sm
*create-story
```

### Product Owner (Validar)
```
@po
*validate-story-draft
```

### Product Manager
```
@pm
*create-epic
```

### DevOps (Git Push)
```
@devops
*push
```

### Analyst (Pesquisa)
```
@analyst
Pesquise alternativas de autenticação
```

### Data Engineer (Banco)
```
@data-engineer
Desenhe a tabela de sessões
```

---

## 📝 Exemplos Rápidos

**Perguntar algo ao Dex:**
```
@dev
Qual é o status do projeto?
```

**Executar comando:**
```
@sm
*create-story
```

**Outro comando:**
```
@qa
*qa-gate
```

---

**Pronto! É isso. Use `@dev`, `@qa`, etc normalmente! 🚀**
